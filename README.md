# PDF Extractor for Colab

[![Open in Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/GladtoDrag/pdf_extract_colab/blob/main/pdf_extract_colab.ipynb)

PDF から本文・表・メタデータを抽出する Colab ノートブックです。

## 使い方
1. 上の Open in Colab をクリック
2. Colab で上から順にセルを実行
3. PDF をアップロード
4. 結果 ZIP をダウンロード
